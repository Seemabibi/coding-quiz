# CODING-QUIZ
* Building a timed coding quiz 

## The Task

To create a timed quiz with multiple-choice questions. 
This will run in the browser and feature dymaically updated HTML and CSS powered by JarvaScript. 

## Acceptance Criteria

```
GIVEN I am taking a code quiz
WHEN I click the start button
THEN a timer starts and I am presented with a question
WHEN I answer a question
THEN I am presented with another question
WHEN I answer a question incorrectly
THEN time is subtracted from the clock
WHEN all questions are answered or the timer reaches 0
THEN the game is over
WHEN the game is over
THEN I can save my initials and my score

```
## Functions 

Includes a multiple-choice option when answering question 
Count down timer located on left side, which counts down once quiz has begun. 
Score is generated and has an option to be saved using initals of user
